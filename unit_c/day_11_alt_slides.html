Two-Phase Framework?

Phase 1: Taking In (Acquisition)

Use when learning something new

Scope First

Break down the topic into sub-topics. Get the map.

Why First

Why does this matter? Where does it fit?

Chunk It

One piece at a time. Don't move on until you get it.

Translate

Ask for analogies, examples, different explanations.

Verify

Cross-check facts. AI can be confident and wrong.

Phase 2: Strengthening (Retrieval)

Use when you have some knowledge to work with

Teach Back

Explain it to AI in your own words. Ask for feedback.

Quiz Me

Ask AI to quiz you. Answer BEFORE seeing feedback.

Minimum Hint

Ask for the smallest possible hint — not the answer.

Socratic Mode

Ask AI to guide you with questions, not explanations.

Topics to Explore

Use the two-phase framework to learn these grid concepts:

  • The Box Model — content, padding, border, margin
  • Layout Grids — columns, margins, gutters
  • 4px Increment System — the implicit grid for spacing
  • Responsive Patterns — 4-column mobile, 12-column desktop
  • Mobile-first vs. Desktop-first — which approach and why

Key question: Am I taking in (new material) or strengthening (testing myself)?

Phase 1: Taking In

If grids are new to you:

  • Scope First — "What are the key concepts I need to understand about grid systems in web/UI design?"
  • Chunk It — Learn one concept at a time (box model, then layout grids, then 4px system)
  • Translate — Ask for real website examples, analogies, visual explanations

Phase 2: Strengthening

Once you have the basics:

  • Teach Back — Explain the box model to AI in your own words. Ask it to check your understanding.
  • Quiz Me — "Quiz me on layout grids and the 4px increment system. Ask one question at a time."
  • Socratic Mode — If something doesn't click, ask AI to guide you with questions.

Apply Your Learning

As you set up grids in Figma, ask yourself:

  • What margins and gutters will you use? Why?
  • How many columns for mobile? For desktop? Why?
  • How does the grid help you place content consistently?

At the end, write a reflection: What did I learn? What's still fuzzy? Which practices did I use?
